# Faris Car v1.0

ESP32-based smart IoT vehicle controlled via the Blynk platform.

## Overview
Faris Car is a personal mechatronics project focused on embedded systems integration,
real-time motor control, and mobile-based user interaction.

## Features
- PWM motor speed control (forward, reverse, left, right)
- WiFi-based control using Blynk (real-time commands and feedback)
- Obstacle avoidance for basic collision prevention
- LED signaling (reverse, turn signals, status)
- 0.96" OLED display for custom messages and status
- Onboard horn functionality via speaker
- Powered by 2 × 3.7V Li-ion batteries
- Cardboard prototype chassis; SolidWorks used for component placement planning

## Hardware
- ESP32 DevKit V1
- L298N motor driver
- 4× DC geared motors (4WD)
- OLED display (0.96")
- Obstacle sensor module
- LEDs + speaker
